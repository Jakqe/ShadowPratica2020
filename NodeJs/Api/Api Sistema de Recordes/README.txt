Desenvolvimento de uma Api para gerenciamento de recordes de jogos utilizando as tecnologias Node.js, MongoDB e Express.

Constituido por duas rotas:
Salvar o primeiro Recorde
Atualizar Recorde

Desenvolvimento de teste automatizado: Sim.
